# SOC Incident Response Lab: Detecting and Investigating a Simulated Ransomware Attack

## 📌 Project Overview

This project demonstrates how to build a Security Operations Center (SOC) lab from scratch using Splunk Enterprise, Sysmon, and Atomic Red Team.

The goal is to simulate a ransomware attack, collect Windows logs, detect malicious activity using Splunk, investigate the attack, and document the complete incident response process.

---

## 🎯 Project Objectives

- Build a Windows security monitoring lab
- Collect Windows logs using Sysmon
- Forward logs to Splunk Enterprise
- Simulate ransomware activity safely
- Create Splunk detection queries
- Investigate the attack
- Produce a professional Incident Response report
- Map attacker behavior to MITRE ATT&CK
