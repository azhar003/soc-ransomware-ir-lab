# SOC Incident Response Lab: Ransomware Detection and Investigation

## 📌 Project Overview

This project documents my journey of building a Security Operations Center (SOC) incident response lab from scratch.

The objective is to understand how security analysts detect, investigate, and respond to ransomware attacks using industry-standard tools and methodologies. Throughout this project, I will simulate ransomware-related activity, analyze security logs, create detection rules, investigate incidents, and document the complete response process.

---

## 🎯 Project Objectives

* Understand the ransomware incident response lifecycle
* Build a SOC lab suitable for my learning environment
* Collect and analyze Windows security logs
* Write detection queries using Splunk SPL
* Investigate simulated ransomware activity
* Map attacker behavior to the MITRE ATT&CK framework
* Create a professional incident response report
* Document the investigation in a structured GitHub repository
